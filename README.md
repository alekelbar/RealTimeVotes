## Run Locally

Clone the project

```bash
  git clone https://github.com/alekelbar/RealTimeVotes.git
```

Go to the project directory

```bash
  cd RealTimeVotes
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn run dev
```
